## [Unreleased]

### Added

- Initial release of Lattice
- Hybrid RAG system combining graph and vector search
- Multi-language parsing support (Python, JavaScript, TypeScript)
- Graph-based structural search with Memgraph
- Vector-based semantic search with Qdrant
- Multiple LLM provider support (OpenAI, Anthropic, Google, Ollama)
- Multiple embedding provider support
- CLI interface with indexing and query commands
- MCP server integration for Claude Code
- Document indexing and drift detection
- Project metadata analysis
- Incremental indexing with content hashing
- Batched graph operations for performance
- Call resolution with type inference
